# 二维梯形蛾眼结构

## 1. 物理对象

显式二维周期梯形结构，用于模拟真实蛾眼表面微结构的减反效果。

## 2. 推荐比较量

- `R(lambda)`
- 周期 `P`
- 高度 `H`
- 顶部占空比 `f_top`
- 底部占空比 `f_bottom`

## 3. COMSOL 导出要求

推荐导出：

- `lambda`
- `abs(ewfd.S11)^2`
- 参数扫描列：`P`、`H`、`f_top`、`f_bottom`

## 4. COMSOL 建模提示

二维周期结构左右边界使用 Floquet 周期条件，上下用端口或散射边界。玻璃基底与梯形结构是否形成联合体，应按材料域是否连续来判断。

## 5. 结果判断

重点比较二维显式结构和一维等效渐变层的反射曲线是否在趋势上吻合。
