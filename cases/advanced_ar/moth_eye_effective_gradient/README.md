# 蛾眼等效渐变层专题

## 1. 物理对象

将蛾眼亚波长结构等效为多层渐变折射率膜，用一维平面模型快速评估减反趋势。

## 2. 推荐比较量

- `R(lambda)`
- 渐变层数
- 顶部和底部有效折射率
- 总厚度

## 3. COMSOL 导出要求

推荐导出：

- `lambda`
- `abs(ewfd.S11)^2`
- 可选：渐变层参数

## 4. Python 运行方式

```bash
python run_advanced_ar_bundle.py
```

## 5. 结果判断

重点说明这是蛾眼结构的等效模型，适合和二维梯形显式结构形成“快筛 + FEM 验证”的关系。
